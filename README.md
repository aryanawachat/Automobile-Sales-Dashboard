# Automobile-Sales-Dashboard
Interactive Automobile Sales Dashboard using Python, Dash and Plotly

## Project Overview
Interactive dashboard analyzing automobile sales trends during recession and non-recession periods.

## Tools Used
- Python
- Dash
- Plotly
- Pandas

## Features
- Yearly Sales Statistics
- Recession Period Analysis
- Interactive Dropdown Filters
- Dynamic Visualizations
